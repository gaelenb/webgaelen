Cowboy Code

My client’s requests:

  • Two sections of text, each with a separate header, that are side-by-side.
  • The width would be no greater than 470px.
  • Responsive at any screen size.
  • As soon as possible.

The caveats:

  • No access to the files (other than the risky admin editor).
  • This was the only content that needed this format.
